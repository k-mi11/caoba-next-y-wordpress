# Paleta de Colores - Tienda Caoba

## Colores Principales
- **Primario**: `#5B3A29` - Café oscuro/terroso (usado en botones, texto principal, navbars)
- **Secundario**: `#E9D8C2` - Crema/beige claro (usado en fondos, botones hover)
- **Acento**: `#ffdd9c` - Dorado/amarillo suave (usado en highlights, logos)

## Colores de Fondo
- **Fondo principal**: `#F7EFE2` - Beige muy claro (home page, sections)
- **Fondo secundario**: `#FFFFFF` - Blanco (cards, productos)
- **Fondo footer**: `#4B2B1B` - Café más oscuro (footer, sections oscuras)

## Colores de Texto
- **Texto principal**: `#2B1A12` - Café muy oscuro (títulos, texto importante)
- **Texto secundario**: `#5B3A29` - Café medio (subtítulos, texto normal)
- **Texto claro**: `#C4AE95` - Beige/grisáceo (footer texto, placeholder)

## Colores de Estados
- **Hover**: `#8f6b4f` - Café medio claro (hover states)
- **Bordes**: `#B58D5E` - Café medio (bordes, dividers)
- **Sombra**: `rgba(91, 58, 41, 0.1)` - Sombra suave del color primario

## Usos Recomendados
- **Headers/Hero**: Fondo `#5B3A29` con texto blanco
- **Botones**: Fondo `#5B3A29` con texto blanco, hover `#8f6b4f`
- **Cards**: Fondo blanco con bordes `#B58D5E`
- **Footer**: Fondo `#4B2B1B` con texto `#C4AE95`
- **Links**: Color `#5B3A29` con hover `#8f6b4f`

## Reglas de Uso
1. **No usar colores fuera de esta paleta**
2. **Mantener consistencia en todo el sitio**
3. **Usar el contraste apropiado para accesibilidad**
4. **El dorado (`#ffdd9c`) solo para elementos especiales**
